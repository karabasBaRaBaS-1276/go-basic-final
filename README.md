# Файлы для итогового задания

В директории `tests` находятся тесты для проверки API, которое должно быть реализовано в веб-сервере.

Директория `web` содержит файлы фронтенда.

# Шаги итогового проекта

1. Инициализируем новый модуль. Для этого в терминале выполнил: 

```
go mod init github.com/karabasBaRaBaS-1276/go-basic-final
```

2. Обновил записимости. Для этого в терминале выполнил:
```
go mod tidy
```
